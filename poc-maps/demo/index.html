üìã About This Demo

This is a proof-of-concept demonstration of 2D tile-based maps for the Adventurer RPG campaign "The Spire of the Lich King". All maps are rendered using pure CSS with no external tileset images required (though they can easily be swapped in).

üé® Implementation Approach:
  • CSS Grid - Fast, responsive, no canvas required
  • CSS Gradients - Procedural textures without image files
  • Unicode Emojis - Interactive elements (player, enemies, NPCs, items)
  • CSS Animations - Water shimmer, void pulse, player glow effects
  • Modular Design - Easy to swap with actual tilesets from OpenGameArt or Kenney

üé® Using Real Tilesets

This demo uses CSS gradients, but you can easily swap to real tileset images:

Recommended Free Sources:

  • OpenGameArt.org - LPC Terrain, Dungeon Tileset (CC0/CC-BY)
  • Kenney.nl - Micro Roguelike, Tiny Dungeon (CC0)
  • Itch.io - 0x72 Dungeon Tileset II (Free)

Implementation with Real Tiles:

.tile-grass {
  background-image: url('/tilesets/grass.png');
  background-size: cover;
  image-rendering: pixelated;
}

.tile-stone-floor {
  background-image: url('/tilesets/stone_floor.png');
}
